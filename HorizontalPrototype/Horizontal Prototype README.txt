README:

The horizontal prototype supports the following features:
- Home page
- Search page
- Simple Filter/Tag system
- More information on dishes
- Instructions for dishes
- Ingredients for dishes
- User log-in page
- Grocery list

The vertical prototype will support:
- Actual searches
- Actual sorting of searches
- Guest Login

The vertical prototype may support:
- Rating dishes
- Rating instructions
- Have view count
- User able to reply to comments
- User able to create comments
- Allow recipe writing (Unlikely)
- More detailed instructions

The vertical prototype will not support:
- An advanced Tag system; users found them cumbersome
- Nutritional information; users not interested
